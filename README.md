# Proyecto Frontend con Vite y Tailwind CSS

Este proyecto es un frontend ligero desarrollado con Vite en modo vanilla y estilizado con Tailwind CSS para crear interfaces rápidas y responsivas.

## Requisitos

- Node.js (version 16 o superior recomendada)
- npm o yarn como gestor de paquetes

## Instalación

Clona el repositorio:

git clone https://github.com/FidelFragoza/rutas_front.git
cd rutas_front

Instala las dependencias:

npm install

## Desarrollo

Para iniciar el servidor de desarrollo:

npm run dev

Esto abrirá una ventana en tu navegador con hot reload para desarrollo rápido.

## Construcción para Producción

Para construir el proyecto listo para producción:

npm run build

Los archivos se generarán en la carpeta `dist` para ser desplegados en cualquier servidor estático.

## Estructura del Proyecto

- `index.html` - HTML base del proyecto.
- `src/` - Archivos fuente JS y CSS.
- `tailwind.config.js` - Configuración de Tailwind CSS.
- `vite.config.js` - Configuración de Vite.
